# Sistema de Gestão de Alunos

Projeto desenvolvido em Java para praticar Programação Orientada a Objetos.

## Funcionalidades

- Cadastro de alunos
- Cadastro de disciplinas
- Inclusão e exclusão de dados
- Listagem de alunos e disciplinas
- Validação simples de acesso ao sistema

## Tecnologias

- Java
- Eclipse IDE
- Git / GitHub

## Conceitos aplicados

- Programação Orientada a Objetos
- ArrayList
- Encapsulamento
- Estruturas condicionais
- Organização de classes
